Understanding Cyber Security Basics & Attack Surface

## Overview
This repository contains Task 1 completed as part of the Cyber Security Internship program.  
The task focuses on building a strong foundation in cyber security concepts such as the CIA Triad, types of cyber attackers, attack surfaces, and awareness of common vulnerabilities.

## Objective
The main objectives of this task are:
- To understand the basics of cyber security
- To learn the CIA Triad with real-world examples
- To identify common types of cyber attackers
- To understand attack surfaces in modern applications
- To gain awareness of OWASP Top 10 vulnerabilities
- To analyze data flow and identify possible attack points

## Task Summary
In this task, I studied core cyber security concepts through self-research and documentation.  
I analyzed commonly used applications such as email, messaging apps, and banking applications to understand how data flows from user to server and where attacks can occur.  
I also reviewed the OWASP Top 10 to understand why common vulnerabilities are dangerous.


## Tools & Resources Used
- Web Browser (Chrome / Firefox)
- OWASP documentation
- Online cyber security learning resources

## Key Learnings
- Understanding of Confidentiality, Integrity, and Availability (CIA Triad)
- Awareness of different types of cyber attackers
- Knowledge of attack surfaces and data flow
- Basic understanding of common web vulnerabilities

## Outcome
This task helped in developing a strong conceptual foundation in cyber security and improved awareness of real-world security risks.
